# 中国象棋 (Xiangqi)

## 游戏简介

中国象棋是一种两人对弈的策略棋类游戏。双方各有16枚棋子，目标是将死对方的将/帅。

## 游戏规则

### 棋盘布局
- 10行 × 9列的棋盘
- 楚河汉界将棋盘分为两半
- 两端各有九宫格（将帅活动范围）

### 棋子类型及移动规则

1. **車（车）**：横竖直线移动，无距离限制
2. **馬（马）**：日字移动，会被蹩马腿
3. **象（相）**：田字移动，不能过河，会被塞象眼
4. **士（仕）**：斜线移动一格，限九宫格内
5. **將/帥（将/帅）**：横竖移动一格，限九宫格内
6. **炮**：隔子吃子，移动时直线无限制
7. **兵/卒**：前进一格，过河后可左右移动

### 特殊规则

- **将帅不能对面**：两将不能在同一列且中间无子
- **将军**：攻击对方的将/帅
- **困毙**：被将军且无法应将时输

## 如何测试

### 1. 访问游戏

启动服务后，访问：
```
http://localhost:51001
```

### 2. 创建房间

1. 登录平台
2. 进入"我的星枢"
3. 选择"中国象棋"游戏
4. 添加玩家（人类或LLM）
5. 配置角色映射：
   - `player_red` → 红方玩家
   - `player_black` → 黑方玩家
6. 点击"播放"开始游戏

### 3. 游戏操作

**人类玩家**：
1. 轮到你时，点击己方棋子
2. 棋盘会高亮显示该棋子的合法移动位置
3. 点击目标位置完成移动

**LLM玩家**：
- 系统自动调用LLM决策并执行移动

## 技术实现

### 行动规范
```typescript
// 固定选项模式，每个合法移动都是一个独立的action
action_id: "move_{fromRow}{fromCol}_to_{toRow}{toCol}"

// 示例：从(9,0)移动到(7,0)
action_id: "move_90_to_70"
```

### 状态结构
```typescript
interface XiangqiState {
  board: (Piece | null)[][];  // 10x9棋盘
  currentRole: 'player_red' | 'player_black';
  players: string[];
  turn: number;
  winner: string | null;
  inCheck: boolean;  // 是否被将军
  lastMove: { from: [number, number]; to: [number, number] } | null;
}
```

### 棋子表示
```typescript
interface Piece {
  type: 'chariot' | 'horse' | 'elephant' | 'advisor' | 'general' | 'cannon' | 'soldier';
  color: 'red' | 'black';
  char: string;  // 中文字符（車、馬、象等）
}
```
