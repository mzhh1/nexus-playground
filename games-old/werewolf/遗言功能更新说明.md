## 功能特性

✅ **触发时机**: 任何死亡（狼人杀害、女巫毒杀、投票放逐、猎人带走）
✅ **多人支持**: 支持多人同时死亡时依次发表遗言
✅ **视觉反馈**: 金黄色主题 + 发光动画，营造紧张氛围
✅ **流程完整**: 遗言后正确处理猎人技能和游戏继续
✅ **观战模式**: 观战者可以看到所有遗言记录

## 使用方法

当玩家进入遗言阶段时：
1. 界面会显示"💬 你即将阵亡，请发表你的遗言。"
2. 在"可用行动"区域会出现"发表遗言"的输入框
3. 输入遗言内容后点击"提交行动"
4. 遗言会被记录并显示在"遗言记录"区域

## LLM玩家支持

✅ **LLM玩家完全支持遗言功能！**

### 信息传递
- 遗言内容通过 `current_state.last_words_history` 传递给LLM
- 包含遗言文本、发表者、死亡原因、时间等完整信息
- LLM在生成提示词时会看到JSON格式的遗言历史

### 记忆更新
- LLM玩家可以通过 `memory_update` 自动记录遗言内容
- 支持在看到遗言后立即更新推理和策略
- 记忆系统已启用（`enable_llm_memory: true`）

### 示例
```json
{
  "last_words_history": [
    {
      "day": 2,
      "speaker": "player_2",
      "content": "我是猎人，3号是狼人！",
      "cause": "vote"
    }
  ]
}
```

LLM玩家可以据此更新记忆：
```json
{
  "memory_update": {
    "mode": "append",
    "content": "第2天：player_2遗言指认3号是狼人，需要重点关注。"
  }
}
```

详细说明请查看：`LLM遗言感知说明.md`
