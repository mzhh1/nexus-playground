# 行动提示配置说明

## 概述

`action-prompts.json` 文件用于配置狼人杀游戏中各个阶段和身份的行动提示信息。这些提示会显示在UI的"可用行动"列表上方，为玩家提供当前阶段的指引。

## 配置结构

### 1. 夜晚阶段提示 (`night`)

夜晚阶段根据子阶段类型显示不同提示：

```json
"night": {
  "guard": { "prompt": "守卫行动提示..." },
  "werewolf": { "prompt": "狼人行动提示..." },
  "seer": { "prompt": "预言家行动提示..." },
  "witch": { "prompt": "女巫行动提示..." }
}
```

### 2. 其他阶段提示

其他阶段使用 `default` 键：

```json
"day_discussion": {
  "default": { "prompt": "白天讨论提示..." }
},
"day_voting": {
  "default": { "prompt": "白天投票提示..." }
},
"last_words": {
  "default": { "prompt": "遗言阶段提示..." }
},
"hunter_shoot": {
  "default": { "prompt": "猎人开枪提示..." }
}
```

## 覆盖的所有阶段

配置文件覆盖了以下所有能具有有效行动列表的阶段：

- ✅ **夜晚 - 守卫阶段** (`night.guard`)
- ✅ **夜晚 - 狼人阶段** (`night.werewolf`)
- ✅ **夜晚 - 预言家阶段** (`night.seer`)
- ✅ **夜晚 - 女巫阶段** (`night.witch`)
- ✅ **白天讨论阶段** (`day_discussion`)
- ✅ **白天投票阶段** (`day_voting`)
- ✅ **遗言阶段** (`last_words`)
- ✅ **猎人开枪阶段** (`hunter_shoot`)

## 如何修改提示

1. 打开 `action-prompts.json` 文件
2. 找到对应阶段的 `prompt` 字段
3. 修改提示文本
4. 保存文件（UI会自动更新）

## 注意事项

- 提示文本应该简洁明了，为玩家提供清晰的指引
- 避免过长的文本，保持在1-3句话
- 确保JSON格式正确，否则会导致加载失败
- 提示文本支持中文和标点符号

## UI显示效果

提示会以蓝色渐变背景框的形式显示在"可用行动"标题下方，样式包括：
- 蓝色渐变背景
- 柔和的边框
- 合适的内边距和圆角
- 舒适的行高和字体大小
